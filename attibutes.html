What are attributes?

An attribute is a customizable value on an html element. They allow us to customize an individual element. Some elements don't function without using certain attibutes.

An element with attributes take the following form.

    <tag attr1="value1" attr2="value2" attr3="value3"></tag>

Images <img>

The <img> element allows us to display images in our pages. We specify what image by giving a url to the src attribute. 
    <img src="assets/hawaii.jpg" width="100%">

Links <a>

The <a> tag has an href attribute that allows us to specify what url to navigate to. You can navigate to pages within your website, as well as ones outside of it.

    <a href="index.html">Table of Contents</a>
    <a href="https://google.com">Google</a>
