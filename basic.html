The paragraph tag <p>

Use this to start a new paragraph. By default, HTML ignores all whitespace (spaces, tabs , newlines ) when it comes to displaying text. This allows you to make your HTML file more readable without messing with the layout of the page. Anything inside a <p>, will appear in the same paragraph.

If you want a new paragraph, you'll need another <p>.

Lists <ol> and <ul>

We use <ul> for an unordered list. For example: 

    <ul>
        <li>Bacon</li>
        <li>Eggs</li>
        <li>Toast</li>
    </ul>

Gives us a bulleted list of breakfast sandwich ingredients.

  • Bacon
  • Eggs
  • Toast

Likewise, <ol> gives use an ordered list.

    <ol>
        <li>Java</li>
        <li>C</li>
        <li>Python</li>
        <li>C++</li>
        <li>C#</li>
    </ol>
  1. Java
  2. C
  3. Python
  4. C++
  5. C#

The <pre> element

The <pre> tag tells the browser to include all whitespace in between the start and end tags. This is great for things like code, poems, movie scripts, etc.

Displaying spreadsheet like data <table>

A lot of web dev is taking spreadsheet style data and displaying it. Sometimes will make it pretty with custom UI, but if we just want the see the data as it is, using a <table> is a good choice.

    <table>
        <thead>
            <th>Title</th>
            <th>Rating</th>
            <th>Box Office Gross</th>
        </thead>
        <tbody>
            <tr>
                <td>Bad Boys For Life</td>
                <td>78%</td>
                <td>$5.8M</td>
            </tr>
            <tr>
                <td>1917</td>
                <td>89%</td>
                <td>$3.6M</td>
            </tr>
            <tr>
                <td>The Voyage of Doctor Dolittle</td>
                <td>17%</td>
                <td>$3.5M</td>
            </tr>
        </tbody>
    </table>
